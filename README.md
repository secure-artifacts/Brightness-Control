# Brightness Control

This lightweight tool lets you easily control the brightness of external displays via your keyboard or the menu bar.


## Key Features

* **Native keyboard shortcut control**  
  Adjust the brightness of external monitors directly using the brightness keys (F1/F2) on your keyboard, without changing your привычный workflow. The app intercepts the input and applies it to your external display in real time.

* **Smooth brightness transitions**  
  Brightness adjustments are no longer abrupt or step-based. Built-in smoothing animation makes transitions feel natural and seamless, similar to the native system experience.

* **Convenient menu bar slider**  
  Click the menu bar icon to bring up a brightness slider. Drag with your mouse for precise control and see the current brightness percentage in real time.

* **Independent multi-display support**  
  Automatically detects all connected external monitors and allows individual brightness control for each one.

* **Smart filtering of built-in display**  
  The app automatically detects and ignores the built-in MacBook display, ensuring your keyboard controls focus only on external screens.

* **One-click launch at startup**  
  Easily toggle launch at login from the menu panel so the control is always available when you need it.


## How to Use

1. **Grant permissions**  
   On first launch, the app will guide you to enable “Accessibility” permissions so it can detect your keyboard input.

2. **Quick adjustment**  
   Press the brightness keys on your keyboard, and the external monitor brightness will change accordingly.

3. **Manual control**  
   Click the “sun” icon in the top menu bar and fine-tune the brightness using the slider.
